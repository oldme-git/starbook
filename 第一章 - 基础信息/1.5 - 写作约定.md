## 命令行
本书会在一些地方使用命令行，我将使用 `$` 符合作为提示符，你不需要输入这个符号。比如，如果我写了 `$ ls`，你只需要输入 `ls` 即可。
```bash
$ echo "Hello, GoFrame!"
Hello, GoFrame!
```
`echo` 是一个输出命令，它会将 `Hello, GoFrame!` 输出到终端。

## 代码省略
为了保证篇幅的整洁，在不必要的竖版代码中会使用`·`来省略代码，比如：
```go
package main

import "fmt"

·
·
·

func main() {
	fmt.Println("Hello GoFrame")
}
```
